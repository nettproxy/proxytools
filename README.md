## Getting Started

### Prerequisites
- **Node.js** (v18.x.x or higher recommended)
- **NPM** (v10.x.x or higher recommended)

### Installation
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/nettproxy/ProxyTools.git
   cd ProxyTools


2. **Install Dependencies:**
    ```bash
    npm install
    ```

### Usage
1. **Run the Server**:
    ```bash
    npm run dev
    ```
